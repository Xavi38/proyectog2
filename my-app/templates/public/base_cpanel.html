Bienvenido Administrador {{ dataLogin.Nombre }}

El propósito principal de este proyecto integrador para el tercer ciclo es diseñar e implementar una maqueta a escala de un Data Center, integrando aspectos de seguridad física y lógica, junto con la conectividad a un servidor de base de datos en la nube.

Materias Involucradas:

  • Seguridad Informática: Responsable de definir las cláusulas de seguridad en el contrato con el proveedor del servicio en la nube y la encriptación de credenciales en la base de datos.
  • Data Center: Encargada del diseño y construcción de la maqueta a escala del Data Center.
  • Electrónica: Implementación de los circuitos y sensores para control de acceso, temperatura, incendios, etc.
  • Gestión de Bases de Datos: Diseño y desarrollo de la base de datos para almacenar la información de usuarios, dispositivos y sensores.
  • Cloud Computing: Desarrollo del servidor y cliente en la nube para acceder a la base de datos y generar códigos de acceso al Data Center.
